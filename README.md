# Aionous (RootAccess)

Phase 1 foundation build for the cyberpunk desktop hacking game.

## Run locally
1. Copy `.env.example` to `.env` and add Firebase values.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start dev server:
   ```bash
   npm run dev
   ```

## Build
```bash
npm run build
```
